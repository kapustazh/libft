# libft

A custom C library that reimplements common libc utilities and basic linked-list helpers.

## Description

`libft` is a foundational 42-style project that collects frequently used string, memory, character-check, conversion, output, and list functions in one reusable static library.

## Build

Run:

make

This creates `libft.a`.

## Clean

Run:

make clean
make fclean

## Usage

Include the header:

#include "libft.h"

Then link your program with `libft.a`.
